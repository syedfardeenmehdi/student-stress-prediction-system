# 🧠 Student Stress Prediction System

## 📌 Overview

The Student Stress Prediction System is a web app. It uses data to guess how stressed students are.

## 🚀 Features

Here are the features:

* The system predicts student stress levels.

* There are modules for admins and students.

* Data can be input and monitored easily.

* The system generates reports.

* It also sends email notifications.

## 🛠️ Tech Stack

The tech stack includes:

* ASP.NET with C#

* SQL Server for database management

* Machine Learning concepts for predictions

## ▶️ How to Run

To run the application follow these steps:

1. Open the project, in Visual Studio.

2. Set up SQL Server.

3. Run the application.

## 🎯 Purpose

The main goal is to help spot and manage student stress with the help of data insights.
